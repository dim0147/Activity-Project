1. Using Microsoft SQL Server Management Studio for importing data. 
2. Open  Microsoft SQL Server Management Studio.
3. On Object Explorer right Databases node.
4. Select Import Data-tier Application.
5. Click Import Settings.
6. Make sure Import from local disk is checked.
8. Name database is Activity.
9. Browse the path and open the data.bacpacfile in Database folder.
10. Click Next and Next to import.
11. Change 'DefaultConnection' and 'DbModel' in Web.config file (please use Visual Studio to change)
